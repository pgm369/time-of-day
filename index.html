Time of Day

A JavaScript Project by Priscila Garcia-Mendoza

Greetings and welcome to my time of day project!

This web page uses JavaScript to determine the current hour, presents a different image, custom message, and color based on each of the following times of day:

  • Morning (5am to 12 noon)
  • Afternoon (12 noon to 6pm)
  • Evening (6pm to 11pm)
  • Night (11pm to 5am)
